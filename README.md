# PRODIGY_WD_03 - TIC TAC TOE WEB APPLICATION

## 📖 Overview
Welcome to the Premium Tic Tac Toe Web Application! This project is a modern, fully responsive, and visually stunning take on the classic Tic Tac Toe game. It was built with a strong focus on UI/UX design, featuring a glassmorphism interface, retro-futuristic synthwave aesthetics, and interactive micro-animations. Whether you want to challenge a friend locally or test your skills against an unbeatable AI, this application provides a seamless and engaging experience.

## 🚀 Live Demo
Experience the game live directly in your browser:  https://varung-coder.github.io/PRODIGY_WD_03/

## ✨ Key Features
- **Dual Play Modes:**
  - **1v1 Mode:** Play locally against a friend on the same device.
  - **Vs AI Mode:** Challenge an unbeatable Minimax algorithm that never loses!
- **Premium Aesthetics:** 
  - A beautiful frosted glass (glassmorphism) game board.
  - Custom generated 3D retro-futuristic gaming background.
  - Eye-catching neon text styling with glowing effects.
- **Fluid Animations:** 
  - Smooth hover scaling, popping placement animations, and a pulsating glow for winning combinations.
  - Floating background particle system for a dynamic feel.
- **Flawless Game Logic:** 
  - Complete turn-based system with win/draw detection.
  - Prevents overwriting filled cells and ignores clicks during the AI's turn.
- **Fully Responsive:** Perfectly scales and maintains spacing across mobile phones, tablets, and desktop monitors.

## 🛠️ Tech Stack
This project was built from scratch utilizing the core foundations of the web:
- **HTML5:** Semantic structure and layout styling.
- **CSS3:** Advanced styling, CSS Grid, Flexbox, keyframe animations, and backdrop filters for glassmorphism.
- **Vanilla JavaScript (ES6+):** Modular game logic, DOM manipulation, Minimax algorithm implementation, and state management.

## 🧠 What I Learned
Throughout the development of this project, I significantly improved my understanding of:
- **Advanced State Management in Vanilla JS:** Keeping the UI perfectly synced with the underlying JavaScript array representing the board.
- **Algorithm Implementation:** Researching and implementing the **Minimax Algorithm** to create a flawless AI opponent that evaluates all possible future moves.
- **Modern CSS Techniques:** Utilizing `backdrop-filter` for glass effects, creating complex gradients, and writing performant `@keyframes` animations.
- **Event Handling:** Ensuring event listeners are managed correctly to prevent bugs, especially when introducing artificial delays for the AI's turn.

## 💻 How to Run Locally
If you wish to run or edit this project on your local machine, it is incredibly straightforward:
1. Clone this repository: `git clone https://github.com/varung-coder/PRODIGY_WD_03.git`
2. Navigate into the folder: `cd PRODIGY_WD_03`
3. Simply open `index.html` in your favorite modern web browser. No complex build tools or servers are required!
